# VoiceLens

## Description
This script uses the Azure AI Service to read the text in front of you using OCR, OpenAI proofreading, and Text to Speech.

## Installation

Follow these steps to install VoiceLens:

1. **Clone the Repository**
   ```bash
   git clone https://github.com/DEmodoriGatsuO/VoiceLens.git
   cd VoiceLens
   ```

2. **Set Up a Virtual Environment (Optional but recommended)**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install Dependencies**
   ```bash
   pip install -r requirements.txt
   ```

After completing these steps, VoiceLens should be installed and ready to use.
